== Description ==

Selecteaza aleatoriu un citat si il afiseaza pe blog sub forma unui widget. ( plugin realizat pentru motivonti.ro )

== Installation ==

Cativa pasi de urmat pentru instalarea plugin-ului:

1. Incarci pe server folderul "citatele-lui-motivonti" in "/wp-content/plugins/"
2. Activezi plugin-ul din meniul "Plugins" al Dashboard-ului
3. Alegi una dintre metode: 
  - daca tema ta nu foloseste Widgets, atunci adaugi codul urmator in sidebar.php: if(function_exists("motivonti")) { motivonti(); }
  - daca tema ta foloseste Widgets, atunci adaugi Widget-ul Motivonti din meniul "Appereance -> Widgets"
